# Generated OpenAPI Snapshots

This folder contains static OpenAPI snapshots for the production-facing services represented by this docs pack.

## Files

- `api-router-openapi.json`
- `cloud-bridge-openapi.json`

## Purpose

Use these files when you need:

- machine-readable schema snapshots for contract checks
- import into API tooling (Postman/Insomnia/OpenAPI generators)
- offline diffing between versions

## Update Source

Snapshots are generated from service runtime code in the source monorepo and copied into this docs repository.
